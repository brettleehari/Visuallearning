How to use

Click any node to cycle its status: not started → in progress → done → not started. When a node moves to done, its downstream nodes become ready automatically. The Ready Now queue is the dev team's morning view — what they can pick up without checking with the PM.
The methodology in one decision: the PM authors the graph; the dev team routes from it. The graph removes the PM as a bottleneck for "what's next?" decisions.
